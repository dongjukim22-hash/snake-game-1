# Snake Game

이 프로젝트는 간단한 웹 기반 Snake(스네이크) 게임입니다.

- **만든 방법:** Helpy Chat(헬피 챗)과 GitHubmcp를 활용하여 자동화된 방식으로 생성 및 업로드되었습니다.
- **구성 파일:** `index.html`, `script.js`, `style.css`

## 실행 방법

1. 이 레포를 클론하거나, 위의 파일들을 다운로드하여 웹 브라우저에서 `index.html`을 실행하세요.
2. 또는, GitHub Pages를 통해 바로 웹에서 플레이할 수 있습니다.

## GitHub Pages로 배포하는 방법

1. **GitHub 저장소에 접속**
   - 예시: https://github.com/dongjukim22-hash/snake-game-1
2. **Settings(설정) → Pages 메뉴로 이동**
3. **Source(소스)에서 Branch를 `main`, 폴더를 `/ (root)`로 선택**
4. **저장(Save) 버튼 클릭**
5. 잠시 후 상단에 배포된 주소가 나타납니다. (예: `https://dongjukim22-hash.github.io/snake-game-1/`)

이제 누구나 웹에서 바로 게임을 즐길 수 있습니다!
